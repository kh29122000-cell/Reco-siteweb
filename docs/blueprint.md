# **App Name**: Filmder

## Core Features:

- Movie Search and Selection: Users can search for a specific movie by name to initiate the recommendation process.
- AI-Powered Movie Recommendations: Integrates the user's KNN machine learning code as a tool to find and present 5 similar films based on a selected movie.
- Tinder-like Interaction: Provides an intuitive 'swipe left for dislike' and 'swipe right for like' interface for movie recommendations, with a 'right swipe' adding the movie to the user's personalized watchlist.
- Personalized Watchlist: Allows users to save movies they have 'liked' to a personal watchlist for future reference.
- Movie Details Display: Displays essential details for each recommended movie, such as title, genre, year, and a brief synopsis.
- Backend Movie Database Integration: Connects to a movie database to retrieve movie names and their associated data required for the KNN algorithm and display.

## Style Guidelines:

- The application employs a dark color scheme to evoke a cinematic, immersive experience.
- Primary color: A rich and vibrant magenta (#CF17CF) to convey passion and modern flair, drawing inspiration from high-contrast film aesthetics.
- Background color: A deep plum-charcoal (#2E1F2E), maintaining the primary hue's essence while providing a sophisticated, low-light setting suitable for content display.
- Accent color: A striking violet-blue (#C775EE) analogous to the primary color, chosen to provide dynamic contrast and highlight interactive elements like swipe indicators or call-to-action buttons.
- Headlines and prominent titles use 'Poppins' (sans-serif) for a modern, geometric, and precise look.
- Body text and detailed descriptions utilize 'Inter' (sans-serif) for its high readability and neutral, objective character, ensuring movie information is easily consumed.
- Bold, clear icons with rounded edges are used for interaction (like/dislike/info), aligning with a 'Tinder-like' feel while maintaining cinematic elegance.
- Centralized, card-based layout for movie recommendations, with prominent artwork and clear overlayed interaction buttons, reminiscent of a dating app interface.
- Smooth and engaging swipe animations for transitioning between movie recommendation cards, providing immediate visual feedback for user interactions.
